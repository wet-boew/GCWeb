Main Container

This designated area serves as the primary HTML container, providing you with the opportunity to seamlessly incorporate all the relevant content for this page.

Main Container with "article"

Article

This represents an article enclosed within the main container.

Main Container with "section"

Section

This represents a section enclosed within the main container.


Code


	<h2>Main Container</h2>
    <main property="mainContentPage">

        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <section> element in this situation] -->
    
    </main>

    <h2>Main Container with "article"</h2>
    <main property="mainContentPage">
        <article>
        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <footer> element in this situation] -->
    
        </article>
    </main>

    <h2>Main Container with "section"</h2>
    <main property="mainContentPage">
        <section>
        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <footer> element in this situation] -->
    
        </section>
    </main>
