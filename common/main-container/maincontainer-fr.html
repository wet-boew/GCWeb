Conteneur Principal

Cet espace désigné sert de conteneur HTML principal, vous offrant la possibilité d'incorporer de manière transparente tout le contenu pertinent pour cette page.

Conteneur Principal avec "article"

Article

Cela représente un article inclus à l'intérieur du conteneur principal.

Conteneur Principal avec "section"

Section

Cela représente une section inclus à l'intérieur du conteneur principal.


Code


	<h2>Conteneur Principal</h2>
    <main property="mainContentPage">

        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <section> element in this situation] -->
    
    </main>

    <h2>Conteneur Principal avec "article"</h2>
    <main property="mainContentPage">
        <article>
        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <footer> element in this situation] -->
    
        </article>
    </main>

    <h2>Conteneur Principal avec "section"</h2>
    <main property="mainContentPage">
        <section>
        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <footer> element in this situation] -->
    
        </section>
    </main>
