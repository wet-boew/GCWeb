CSS Class

.p-0
Set padding to 0 for the element
.pl-2
Set left padding of 2 for the element
.mt-auto
Set auto margin-top for the element
.mb-sm-5
Set bottom margin of 5 for the element on small screens
.p-sm-3
Set padding of 3 for the element on small screens
