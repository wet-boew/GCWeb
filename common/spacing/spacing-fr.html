Classe CSS

.p-0
Définir un rembourrage de 0 pour l'élément
.pl-2
Définir un rembourrage à gauche de 2 pour l'élément
.mt-auto
Définir une marge supérieure automatique pour l'élément
.mb-sm-5
Définir une marge inférieure de 5 pour l'élément sur les petits écrans
.p-sm-3
Définir un rembourrage de 3 pour l'élément sur les petits écrans

Apparence

Exemple 1: Pas de rembourrage

Exemple 2: Rembourrage à gauche de 2

Exemple 3: Marge supérieure automatique

Exemple 4: Marge inférieure de 5 sur les petits écrans

Exemple 5: Rembourrage de 3 sur les petits écrans

Code

// Exemple 1:
<p class="p-0">...</p>
// Exemple 2:
<p class="pl-2">...</p>
// Exemple 3:
<p class="mt-auto">...</p>
// Exemple 4:
<p class="mb-sm-5">...</p>
// Exemple 5:
<p class="p-sm-3">...</p>
