This is an example test for comparative implementation purposes and for tracking coexistence changes across versions. In particular, this example shows a width inconsistency that is only visible in small viewports

Technical notes

To reproduce the concern:

  • This happen when the attribute place-items="center" are set on the GCDS grid
  • Images width need to lower than the width available in the view port
  • The description text take less width space than the width available in the view port

Workaround, at least one of the following:

  • Use images that have a minimum width of 400 pixels
  • Ensure the description text will take at least 2 lines, like with a least 70 characters
