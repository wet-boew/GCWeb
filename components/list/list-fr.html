Listes en colonnes .list-col-[breakpoint]-[columns]

Ajoutez la classe list-col-[breakpoint]-[columns] pour afficher n'importe quelle liste dans un format de colonne, où [breakpoint] peut être xs, sm, md ou lg et [columns] peut être soit 1, 2, 3 ou 4.

Le plus petit point d'arrêt défini affectera son point de rupture et tous les points d'arrêt plus grands. Par exemple, list-col-sm-2 affichera la liste en deux (2) colonnes pour les points d'arrêt sm, md et lg. Le nombre de colonnes par défaut est un (1), donc, dans cet exemple, la liste serait affichée dans une colonne pour le point d'arrêt xs.

Les classes peuvent être combinées pour afficher les éléments dans un nombre différent de colonnes pour différents points d'arrêt. Voir l'exemple suivant.

Afficher les éléments verticalement .list-vertical

Par défaut, les éléments de la liste sont disposés horizontalement (de gauche à droite) à l'aide de Flexbox. Pour afficher les éléments verticalement (de haut en bas), il suffit d'ajouter la classe list-vertical à l'élément de liste. Cela permet de passer à une mise en page utilisant des colonnes CSS au lieu de Flexbox.
