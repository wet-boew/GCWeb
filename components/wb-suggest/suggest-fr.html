Chargement de suggestion pour un datalist avec un fichier JSON.

Exemple

Exemple avec un élément <template>

Propriété configurable

Note: Afin de réduire le nombre de requête HTTP, le fichier JSON contenant les suggestions est seulement téléchargé lorsque le champs reçois le focus.

La configuration est effectué via des data attribute défini sur l'élément datalist.

data-wb-limit
Nombre maximal de suggestion à afficher
data-wb-filter-type
  • any (défault), n'importe où
  • startWith débutant par
  • word correspondant à un mot
data-wb-suggest
L'URL du fichier JSON avec un pointeur JSON vers le tableaux de suggestion. Par exemple: data/list.json-ld#/suggestions

Événement

wb-update.wb-datalist
Déclanché lorsque les suggestions dans le datalist a été mise à jour. Principalement utilisé pour une intégration avec le polyfill Datalist.

Exemple de fichier JSON-LD

{
	"@context":{
		"dc": "http://purl.org/dc/terms/",
		"suggestions": {
			"@id": "http://www.w3.org/2001/XMLSchema#string",
			"@container": "@list"
		}
	},
	"@language": "fr",
	"dc:title": "Titre de votre fichier contenant des termes suggéré",
	"dc:modified": "2021-04-23",

	"suggestions": [
		"Mot suggéré 1",
		"Mot suggéré 2",
		"Mot suggéré 3",
		"Mot suggéré 4"
	]
}
