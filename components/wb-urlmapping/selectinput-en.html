Select inputs by default based on url query string.

  • Empty ?
  • Bananas ?f=b"
  • Grapes ?f=g"
  • In English in Ottawa ?lng=eng&loc=otw"

  • lng=fr for French
  • lng=eng for English
  • loc=mtl for Montreal
  • loc=otw for Ottawa
  • loc=tor for Toronto
