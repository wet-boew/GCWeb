Campaign Menu

Example of a new menu that uses megamenu for md and lg, and GCWeb menu for sm and xs.

This méli-mélo project is sponsored by PCH department.

  • Home - demo the default menu colour which is the bg-gctheme colour
  • About Winterlude - demo wb-navcurr for a sub menu page, and a black menu colour
  • Calendar - demo for when global Canada.ca GCWeb menu is already present, this custom menu will not be activated by the plugin

Winterlude Site Menu

  • Home
  • Calendar
  • About

Implementation plan

(To be revised during fall 2024)

  • Stabilize the plugin amalgamation - estimated by May 2024
  • Revise the color for a better alignment with the Canada.ca default color scheme - estimated by Summer 2024
  • Collect user experience feedback on this redesigned menu after the Canada Day 2024 event - Report summary to added by Summer 2024
  • Make the code independent from the wet-boew mega menu and from the GCWeb menu - Fall 2024
  • Engage with DTO and web apps community (ex. CDTS implementer) about this initiative - Now to fall 2024
  • Work toward a provisional plugin - Autumn 2025
