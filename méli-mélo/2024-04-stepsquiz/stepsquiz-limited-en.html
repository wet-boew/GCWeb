Quiz pattern for the Steps Form component using the panel class along with the content limited width template.

How to use

  • Add the CSS class name cnt-wdth-lmtd to the <body> of your page.
  • Apply panel, panel-default and stepsquiz CSS classes to the first <section> element of the Steps Form.
  • Add the panel-body CSS class to the <div> element containing the wb-steps class.
