La première version (v1) de l'entête est obsolète mais toujours supportée.

Version 2.0

Cette version requière l'implémentation de la version 2 du menu GCWeb ou le code <div class="container"> a été relocalisé dans l'entête donc l'extérieur du composant "gcweb-menu".

Entête

<header>
	<div id="wb-bnr" class="container">
		<div class="row">
			<!-- Changer de langue [version 1] -->
			<!-- Image de marque [version 1] -->
			<!-- Recherche [version 1] -->
		</div>
	</div>
	<hr>
	<div class="container">
		<div class="row">
			<div class="col-sm-12">
				<!-- Menu du site [version 2] -->
			</div>
		</div>
	</div>
	<!-- Fil d'Ariane [version 1] -->
</header>
