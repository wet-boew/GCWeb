Pour afficher seulement le bouton "Partager cette page" sans le bouton "Signaler un problème sur cette page", créez une <div> avec l'attribut de données suivant :data-wb-share='{"lnkClass": "btn btn-default btn-block"}' et la classe CSS suivante class="wb-share"

Pour plus d'informations sur le plugin Partager cette page, veuillez visiter la page du Gadget de partage

Code HTML

Ce code html générera la même section de détails de page que celle illustrée ci-dessous 

<footer class="pagedetails">
	<h2 class="wb-inv">Détails de la page</h2>
	<div class="row">
		<div class="wb-share col-sm-4 col-md-3 col-sm-push-8 col-md-push-9" data-wb-share='{"lnkClass": "btn btn-default btn-block"}'></div>
		<div class="col-sm-6 col-md-5 col-lg-4 col-sm-pull-4 col-md-pull-3">
			<dl id="wb-dtmd">
				<dt>Date de modification :</dt>
				<dd><time property="dateModified">2022-04-14</time></dd>
			</dl>
		</div>
	</div>
</footer>
